I worked as a Software Engineering Intern at BuzzFeed London between June 3rd 2019 - August 23rd 2019. 
I worked on many projects which allowed me learn and work on my coding and programming skills, my designs and researching for different layouts and potential features.

BuzzFeed Internet Points

During my internship as a software engineer at BuzzFeed, one of my projects was to implement the new BuzzFeed ‘Internet Points’ system and badge on to BuzzFeed.com. I used HTML, CSS, BuzzFeed’s CSS library ‘Solid’ and had to work with the different APIs to create the points badge with updated points and the tooltip (which was also on a timer so users can comfortably read the tooltip message and click the link).
The points badge is located on the different user’s profile pages and they collect different points via different triggers. The number of users that started interacting and signing up to BuzzFeed community increased after our points system launched.


BuzzFeed Trophy Case

Researched potential trophy case ideas and coded the different icons, layouts and features designs to allow the team to see the pros and cons of each design idea.


BuzzFeed HackWeek Game

For BuzzFeed’s Hack Week I created the game based on what was a popular topic with BuzzFeed readers (Queer Eye) and created a game which user can play the different characters and collect different items, through different sceneries and it was essentially a game with 5 different personalised games based on different Queer Eye characters. I worked with a designer from BuzzFeed who created the art work for the different versions for the game.  (The game works by using the left, right and spacebar keys).
